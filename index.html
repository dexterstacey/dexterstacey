Reframe.network

I built a profile site called Reframe that showcases the work of refugee-led organisations and helps them connect with donors. It's built using the LAMP stack and hosted on an AWS EC2 instance.

Front-end Tech Used: Javascript, HTML, CSS

Back-end Tech Used: PHP, MariaDB, Apache, virtual Debian instance on Amazon EC2.


Since 2014 I have worked with Xavier Project, a charity based in East Africa that supports refugee-led organisations.

As well as creating the Reframe website, I have led the development of Xavier Project's overall operational model which enfranchises refugees to lead the international aid and development process in their communities. So far this model has helped 20 RLOs reach more funding from international actors and deliver better quality projects to over 12,000 children and adults.
